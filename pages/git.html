Hướng dẫn sử dụng Github

1. Chuẩn bị.

- Các bạn đăng kí 1 tài khoản github

- Các bạn đăng kí 1 tên miền trên trang Dot.tk(Lưu ý: khi tìm kiếm tên miền không nhập ".tk" vào cuối)

2. Phần liên kết.

- Setting trên Domain:

+ Đầu tiên bạn đăng nhập vào trang my.freenom.com bạn nào đã đăng kí thì chắc chắn biết trang này.:))

Trang chủ--> Service --> My Doamains

Tiếp theo. ở dòng tên miền của bạn click Manage Domain

Chọn ->Manage Freenom DNS

Tiếp theo chọn Add Records

Dòng đầu tiên bạn chỉ nhập vào Target Ip 185.199.108.153 đây là ip của github page, bạn muốn tìm hiểu thêm có thể search google. 2 ô trước để trống

Tiếp: Add Records

+ Ở trường Name bạn nhập vào www.

+ Ở trường Type bạn nhấn vào chọn thẻ CNAME.

+ Ở trường Targer bạn nhập vào tên domain bạn vừa đăng kí. Vd(baonn.tk) sau khi nhập xog nó sẽ như này.

Nhớ nhấn SAVE CHANGES sau mỗi lần thêm.

- Setting trên github

+ Sau khi đăng kí tạo mới 1 Repository bạn nào có rồi thì kéo xuống đọc tiếp:

Trường Name nhập tên gì cũng được.

Làm theo ảnh nha.

Tiếp nhấn vào Setting

Kéo xuống phần GitHub Pages

Click vào None chọn master Branch

Sau khi load lại trang thì bạn kéo xuống lạiGitHub Pages

Ở ô Custom domain bạn nhập tên Domain lúc nãy đăng kí vào và nhấn Save

Ok Vậy là xong phần setting nhé. Chờ khoảng tầm 5-10p vào ô tìm kiếm gõ tên miền của bạn và tận hưởng.

Còn làm sao để up project của bạn lên github thì google nha. Bạn mình hay xài Gitdesktop còn nhóm mình thì không xài cái đó.

Có nhiều cách lắm chịu khó Google đi.
