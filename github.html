Hướng dẫn sử dụng Github

1. Chuẩn bị:
- Các bạn đăng kí 1 tài khoản Github.
- Các bạn đăng kí 1 tên miền trên trang Dot.tk.
2. Phần liên kết:
2.1 Thiết lập Domain:
- Đầu tiên, bạn đăng nhập vào trang my.freenom.com:
+ Chọn Service tiếp theo chọn My Domains
+ Tiếp theo, bạn chọn Manage Domain
+ Chọn Manage Freenom DNS.
+ Tiếp đến, chúng ta bắt đầu thiết lập Domain:
- Ở Target chúng ta thiết lập Ip 185.199.108.153 đây là ip của github.
- Trường Name bạn nhập www.
- Trường Type bạn chọnCNAME.
- Trường Targer bạn nhập vào tên domain bạn vừa đăng kí. Vd: (hci2019group08.tk)
- Nhấn Save Changes.
2.2 Thiết lập Github:
- Sau khi đăng nhập, ta chọn New repository:
- Tiếp theo, chúng ta chọn Settings lướt xuống GitHub Pages
Vậy là xong phần Settings nhé. Chờ khoảng tầm 5-10p vào ô tìm kiếm gõ tên miền của bạn và tận hưởng thành quả.

Chúc các bạn thành công!
